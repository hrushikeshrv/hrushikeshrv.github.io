Designed specifically for serving low traffic with low compute power servers.

docxlatex

docxlatex is a python module for extracting text from .docx files, including support for mathematical and chemical equations, primarily for use with tools like MathJax. It also includes support for extracting images and conversion of extracted equations into LaTex syntax. source code documentation

MJXGUI

MJXGUI is an equation editor for the web for building mathematical and chemical equations with a GUI similar to Google Docs or Microsoft Word. It supports conversion of the entered equations into LaTeX for storage and rendering using tools like MathJax. Try the demo below and create an equation. MJXGUI is still in beta and does not support all of its features on mobile devices yet. source code documentation demo

LifeNotes

LifeNotes is a timeline web-app that allows you to record key moments in your life with specific "notes". You can give notes different tags to sort them and find them faster. It shows you an interactive timeline with all the notes you want to see so you can visualize your progress and easily keep track of events. The LifeNotes timeline works best on desktop devices or big screens. notes.hrus.in Demo
